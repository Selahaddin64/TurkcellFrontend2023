    listings, etc.</td>
  </tr>

    <td>Defines a footer for a document or section</td>
  </tr>
  <tr>
    <td>&lt;header&gt;</td>
    <td>Specifies a header for a document or section</td>
  </tr>
  <tr>
    <td>&lt;main&gt;</td>

  </tr>
  <tr>
    <td>&lt;mark&gt;</td>
    <td>Defines marked/highlighted text</td>
  </tr>
  <tr>
    <td>&lt;nav&gt;</td>

  </tr>
